# XTools

## 一个开源的跨平台工具箱。

## 专注工具，助力高效。

## 完全本地化工具，无需云端，不担心隐私泄露。

## 免费开源，无广告，无推广。

## 现阶段实现功能

- JSON格式化
- JSON对比
- 图片分割
- 图片压缩
- 图片识别文字
